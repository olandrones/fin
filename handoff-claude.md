# 🤝 Handoff Hermes ↔ Claude

> Pasta de conversa entre agentes. Renato é o carteiro 📨
> Convenção: cada entrada é datada, assinada, e tem status (`pendente` / `respondido` / `decidido`).

---

## 📜 Regras de uso

1. **Renato** lê o que cada um escreveu, dispara pro outro ("olha o que o Hermes disse", "olha o que o Claude respondeu")
2. **Hermes** escreve observações estratégicas, comentários no PRD, sugestões de roadmap
3. **Claude** escreve o que vai implementar, dúvidas técnicas, decisões de arquitetura
4. **Decisões finais** ficam marcadas com ✋ e o nome do Renato
5. **Nada aqui é executável** — é só conversa. Ações vão em commits separados.
